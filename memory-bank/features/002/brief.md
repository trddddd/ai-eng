# Brief: Content Bootstrap for Study MVP

**Issue:** trddddd/ai-eng#2
**Author:** @trddddd (Roman Kirpichnikov)
**State:** CLOSED

## Проблема

После регистрации пользователь попадает в пустую систему: нет лексем, нет предложений, нет переводных связей. Учебный контент отсутствует полностью. Следующая фича — персональные карточки и study flow — не может быть реализована без наполненного каталога.

## Для кого

Конечный пользователь платформы (learner), для которого следующая фича создаёт персональные карточки поверх каталога.

## Откуда взялась задача

Проект инициализирован: Rails 8, регистрация, FSRS gem подключён. Доменная модель для контента ещё не существует. Без неё невозможно двигаться к следующим фичам.

## Что мы хотим получить

После `db:setup` проект содержит непустой каталог контента — все лексемы из Oxford 5000, NGSL Core и NGSL Spoken:

- лексемы с переводами
- предложения-примеры с исходным текстом без UI-плейсхолдеров
- переводные связи между предложениями
- вхождения целевых лексем в предложениях

Домен поддерживает несколько языков.

## Вне скоупа

Дизайн карточек, study flow, UX онбординга, аудио.
