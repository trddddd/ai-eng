# Brief: Персональная стартовая колода после регистрации

**Issue:** trddddd/ai-eng#10
**State:** OPEN

## Проблема

После signup пользователь попадает в пустой аккаунт — нет ни одной карточки, доступной для изучения. Первый сеанс работы с продуктом невозможен без ручной настройки: нужно выбрать набор, разобраться в интерфейсе и запустить процесс самому. Продукт не работает сразу после регистрации.

## Для кого

Новый пользователь (learner) — только что зарегистрировался, мотивация высокая, терпение низкое. Ожидает начать учиться немедленно, без онбординга.

## Откуда взялась задача

Каталог лексем и предложений загружен (фичи #2, #4). FSRS-алгоритм готов к использованию. Механизм карточек существует. Но все эти компоненты не связаны с конкретным пользователем в момент его первого входа в продукт — пользователь оказывается перед пустым экраном.

## Что мы хотим получить

Сразу после регистрации пользователь видит готовую колоду карточек из набора по умолчанию и может немедленно начать первую учебную сессию. Каждая карточка имеет персональное FSRS-состояние, привязанное к этому пользователю.

## Вне скоупа

Выбор набора слов в процессе самой регистрации (onboarding flow), редактирование/удаление стартовой колоды, поддержка нескольких наборов одновременно при signup.
