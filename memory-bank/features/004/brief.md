# Brief: Sentence Domain & Quizword Import

**Issue:** trddddd/ai-eng#6
**State:** OPEN

## Проблема

Каталог лексем существует — слова, уровни, переводы загружены. Но учебных примеров нет: ни одного предложения, ни одной привязки к слову в контексте. Без предложений невозможно создавать cloze-карточки для учебного процесса.

## Для кого

Learner — пользователь, который учит слово в контексте: видит предложение с пропуском и вводит нужную форму.

## Откуда взялась задача

Фича #2 закрыта: `Language`, `Lexeme`, `LexemeGloss` загружены в БД. Следующий логичный шаг по домену — предложения, привязанные к лексемам. Без этого шага невозможно двигаться к учебному процессу.

## Что мы хотим получить

В БД появляются предложения на английском с переводами, каждое привязано к хотя бы одной лексеме из каталога.
