## Product Context

**Lingvize** — веб-платформа для изучения иностранных языков через интервальное повторение и карточки.

Пользователи учат слова в контексте предложений: у слов есть примеры из внешних корпусов (Tatoeba, Quizword), учебные карточки строятся из этого контента по принципу cloze deletion (пропущенное слово в предложении). Ответы и история повторений сохраняются для планирования по алгоритму FSRS.

Система подстраивает сложность и время следующего повторения по ответам пользователя и поведению в сессии (скорость, точность, количество попыток). Есть генерация аудио, подбор примеров из внешних корпусов и сервисов, аналитика обучения.

Существующие решения (Anki, Quizlet) требуют от пользователя самостоятельно создавать контент или пользоваться чужими наборами без контроля качества. Lingvize даёт готовый, курируемый контент с привязкой к CEFR-уровням и автоматическим подбором примеров.

Интерфейс многоязычный. Языком разработки и продуктовых текстов по умолчанию считается русский.

## Core Workflows

- `WF-01` **Регистрация и онбординг** — пользователь регистрируется, автоматически получает стартовую колоду из 50 карточек уровня A1 с русскими переводами.
- `WF-02` **Сессия повторения** — пользователь открывает /review, видит cloze-карточку, вводит пропущенное слово, система оценивает ответ (Levenshtein distance), классифицирует recall quality, применяет FSRS для планирования следующего повторения.
- `WF-03` **Контент-пайплайн** — импорт лексем (Oxford 5000, NGSL), импорт предложений и переводов (Quizword), матчинг предложений к лексемам, формирование SentenceOccurrence для cloze-карточек.
- `WF-04` **Админка контента** — работа с контентом: слова, предложения, курсы и карточки, сценарии онбординга контента.

## Outcomes

| Metric ID | Metric | Baseline | Target | Measurement method |
| --- | --- | --- | --- | --- |
| `MET-01` | Каталог лексем с переводами | 0 | 9500+ (Oxford + NGSL) | `Lexeme.count` + `LexemeGloss.count` |
| `MET-02` | Предложения с привязкой к лексемам | 0 | 10000+ | `SentenceOccurrence.count` |
| `MET-03` | Стартовая колода после регистрации | Пустой аккаунт | 50 карточек A1 | `Card.where(user:).count` после регистрации |
| `MET-04` | Адаптивное повторение по FSRS | Нет | Полный цикл: ответ → оценка → перепланирование | Ручная проверка flow + RSpec |
| `MET-05` | Время cold start (bin/setup) | ~30 мин | ≤5 мин | Замер с pg_dump/pg_restore |

## Constraints

- `PCON-01` Первичные ключи — всегда UUID v7.
- `PCON-02` Не менять существующие миграции задним числом; новые изменения схемы — только новыми миграциями.
- `PCON-03` PostgreSQL доступен только через Docker Compose, порт 5433.
- `PCON-04` Не подключать новые гемы и интеграции без явного запроса.
- `PCON-05` Файлы и именование совместимы с Zeitwerk.
- `PCON-06` Язык UI и продуктовых текстов — русский (i18n: ru primary, en fallback).
