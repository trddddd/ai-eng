# Prompt Index

## Brief / Feature

- [feature/brief.md](feature/brief.md) — создать transient brief draft и GitHub Issue.
- [feature/brief-review.md](feature/brief-review.md) — review brief через отдельную роль.
- [feature/review.md](feature/review.md) — strict review canonical `feature.md`.
- [feature/create-attempt.md](feature/create-attempt.md) — создать и выполнить attempt в worktree.
- [feature/attempt-review.md](feature/attempt-review.md) — проверить attempt перед accept/revise.
- [feature/resume.md](feature/resume.md) — продолжить FT-XXX по state machine.

## Plan / Eval / Workflow

- [plan/review.md](plan/review.md) — strict review `implementation-plan.md`.
- [eval/create-suite.md](eval/create-suite.md) — создать eval suite из `feature.md`.
- [eval/run.md](eval/run.md) — выполнить eval suite и принять accept/revise/escalate.
- [workflow/eval.md](workflow/eval.md) — workflow-level проверка trajectory, handoff и gates.

## ADR / PRD

- [prd/review.md](prd/review.md) — PRDAS review.
- [adr/create.md](adr/create.md) — создать ADR для blocking decision.
