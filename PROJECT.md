# Lingvize

**Lingvize** — веб-платформа для изучения иностранных языков через интервальное повторение и карточки.

Пользователи учат слова в контексте предложений: у слов есть примеры, учебные карточки строятся из этого контента, ответы и история повторений сохраняются для последующего планирования.

Система подстраивает сложность и время следующего повторения по ответам пользователя и поведению в сессии. Есть генерация аудио, подбор примеров из внешних корпусов и сервисов, аналитика обучения.

Интерфейс многоязычный. Языком разработки и продуктовых текстов по умолчанию считается русский.

В продукте также есть админка для работы с контентом: слова, предложения, курсы и карточки, а также сценарии онбординга контента — наполнение базы примерами и привязка предложений к словам, у которых ещё нет готовых учебных карточек.
