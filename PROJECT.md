# Lingvize

**Lingvize** — веб-платформа для изучения иностранных языков на основе интервального повторения и карточек. Пользователи работают со словами в контексте предложений: для слов хранятся примеры, формируются учебные карточки, фиксируются ответы и повторения.

Система учитывает ответы и ход сессии, чтобы подстраивать сложность и момент следующего показа карточки. Есть озвучка материалов, подбор примеров из внешних источников и аналитика обучения. Интерфейс многоязычный; язык разработки и текстов по умолчанию — русский.

Для сопровождения контента предусмотрена **админ-панель**: управление словами, предложениями, курсами и карточками, а также сценарии онбординга контента — последовательное наполнение базы предложениями и привязка примеров к словам без готовых карточек, подготовка материалов для обучения.
